Black holes seem very complicated, but they obey a set of simple rules. These rules are, in fact, just the laws of thermodynamics in disguise. By exploiting them, we can explain the basic ideas behind some modern gravitational wave observations using only concepts from a first course in thermal physics. This allows research topics to be discussed in introductory physics lectures.
